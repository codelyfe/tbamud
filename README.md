## How to Compile and run on Ubuntu 16.04 amd64 

Step 1
```
cd /file/path/
```
Step 2
```
sh ./configure
```
Step 3
```
cd /file/path/src
```
Step 4
```
sh ./licheck
```
Step 5
```
make all
```
Step 6
```
cd ..
```
Step 7
```
sh ./autorun
```
